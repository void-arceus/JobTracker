# 🚀 Job Tracker App

A full-stack Job Tracker application to manage and organize your job applications efficiently.
Built with a clean UI, modern stack, and deployed for real-world usage.

---

## ✨ Features

- 🔐 Authentication (Login / Register)
- 📊 Dashboard with job summary
- ➕ Add / ✏️ Edit / ❌ Delete jobs
- 🏷️ Job status tracking (Applied, Interview, Offer, Rejected)
- 🔍 Filter & Search functionality _(in progress / optional)_
- 📱 Fully responsive UI (mobile-first)
- 🌐 Deployed frontend & backend

---

## 🛠️ Tech Stack

### Frontend

- React (Vite)
- Tailwind CSS
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication

## ⚙️ Environment Variables

### Frontend (`.env`)

```
VITE_API_URL=https://your-backend.onrender.com
```

### Backend (`.env`)

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLIENT_URL=https://your-app.vercel.app
```

## 🚀 Getting Started Locally

### 1️⃣ Clone the repo

```
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm run dev
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```
